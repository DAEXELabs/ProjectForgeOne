# Forge One OS

Forge One OS is the operational intelligence platform of Forge One Infrastructure.

Its purpose is to help businesses remember more, think faster, automate repetitive work, and make better decisions.

## Core Mission

Build the infrastructure intelligent businesses will depend on.

## Initial Focus

Version 0.1 will focus on:

- Home Dashboard
- Entity Workspace
- Knowledge Vault
- Forge Assistant
- Documents
- Proposal Service

## Status

Foundation phase.
